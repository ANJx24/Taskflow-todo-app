# 🚀 TaskFlow - Modern To-Do App

A modern and responsive task management application built with **React.js** and **Vite**. TaskFlow helps users organize tasks, manage deadlines, track progress, and improve productivity through a clean and intuitive interface.

---

## ✨ Features

### 📝 Task Management

* Add New Tasks
* Edit Existing Tasks
* Delete Tasks
* Mark Tasks as Completed
* Undo Completed Tasks

### 📅 Task Scheduling

* Set Due Date
* Set Due Time
* Priority Levels

  * 🔴 High
  * 🟡 Medium
  * 🟢 Low

### 📊 Dashboard

* Total Tasks Counter
* Completed Tasks Counter
* Pending Tasks Counter
* Productivity Percentage

### 🔍 Search Functionality

* Real-time Task Search
* Instant Filtering

### 💾 Local Storage

* Tasks persist after browser refresh
* No backend required

### 📱 Responsive Design

* Mobile Friendly
* Tablet Friendly
* Desktop Friendly
